Bar Charts

Used to compare quantities across different categories. The length of the bar corresponds to the value of the category being measured. Bar charts are useful for comparing discrete data points.

Line Charts

Ideal for displaying data points over time. A line connects individual data points, making it easy to see trends and patterns. These are commonly used for time-series analysis to track changes over periods.

Pie Charts

Represent proportions of a whole. Each slice corresponds to a category, and the size of each slice is proportional to its value. Pie charts are often used for showing percentage breakdowns.

Scatter Plots

Display the relationship between two continuous variables. Each point represents a data point on the x and y axes. Scatter plots are useful for identifying correlations or patterns between variables.

Area Charts

Similar to line charts, but with the area beneath the line filled with color to emphasize the magnitude of values. They are useful for showing trends over time with an emphasis on volume.

Histograms

A type of bar chart used to represent the frequency distribution of a set of continuous data. It is ideal for understanding the distribution and spread of numerical data.

Radar Charts

Also known as spider or web charts, these are used to display multivariate data in a way that allows comparison between several variables. Radar charts are ideal for comparing data with different units.
